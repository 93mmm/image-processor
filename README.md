image processor
===

#### How to compile?

type in terminal these commands:

```bash
mkdir build
cd build
cmake ..
make
mv image-processor-tg-bot ../env
cd ../env
```
or 
```bash
chmod +x bash.sh
./bash.sh
```

#### How to run?
```bash
./image-processor-tg-bot
```

stb_image and stb_image_write the files are taken from the repository https://github.com/nothings/stb
